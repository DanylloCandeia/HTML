<hr>.............: Cria linha horizontal.
<p></p>..........: Paragrafo.
<b></b>..........: Negrito (não semântica).
<strong></strong>: Negrito (semântica).
<i></i>..........: Itálico (não semantica).
<em></em>........: Itálico (semântica).
<mark></mark>....: Marca texto.
<small></small>..: Palavra pequena.

<ins></ins>......: Sublinhado.

<sup></sup>......: Texto sobrescrito EX.: númeri elevado.
<sub></sub>......: Texto subscrito.

<pre></pre>......: Mantem a mesma identação da IDE.
<q></q>..........: Acrescenta aspa dupla(usado para citação simples)

<ol></ol>........: Lista ordenada.
<ul></ul>........: Lista não-ordenada.
<li></li>........: Lista de elementos de uma lista.
<dl></dl>........: Lista de definição 
<dt></dt>........: Termo
